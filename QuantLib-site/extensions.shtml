QuantLib Extensions

QuantLib is available as a C#, Guile, Java, MzScheme, Perl, Python, and Ruby module by means of SWIG.
Compilation of the Java module is performed by means of sKWash. A guide to the process can be found here.
The SWIG interfaces are available in the QuantLib CVS repository in the QuantLib-SWIG module and can be used as a starting point for exporting QuantLib towards other languages.

QuantLibAddin exports a procedural interface to a number of platforms including Microsoft Excel and OpenOffice.org Calc.

QuantLibXL is an initial Excel add-in built using xlw.

Initial GNU R support is provided by means of RQuantLib by Dirk Eddelbuettel.

Niels Elken Sønderby has a page for the tentative Mathematica project QuantLibMma.

There is a strong interest in having QuantLib available as extension for different platforms. We're looking for volunteers to help with:

  • COM for Microsoft world (VB and Excel);
  • Gnumeric add-in;
  • Matlab/Octave extension.
  • Mathematica extension.
  • S-Plus extension.
  • CORBA
  • Simple Object Access Protocol (SOAP)
  • XML financial instrument definition (FpML?)
Interested developers can post on the QuantLib-dev mailing list.
