Download Page

The latest QuantLib release is the 0.3.8 version (December 2004.)

Debian users may find version 0.3.8 of the different QuantLib packages in the "unstable" (a.k.a. "Sid") distribution; see the Debian site for a list of available packages.

For Mac OS X users, Fink packages are available in the "10.3/unstable" tree; for a list of available QuantLib-related files see the Package Database. To enable the "unstable" tree, please read the Fink faq. Be sure to run "fink selfupdate" to obtain the latest package definitions, and then run "fink install quantlib" to install the main library.

Packages for other platforms are available from the SourceForge download page in a number of different formats. This also applies to eager Debian or Mac OS X users not wanting to wait for the library to be packaged, or to advanced users wanting to configure the library with non-default settings.
